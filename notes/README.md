# notes/

Active scratch space only. Nothing here is loaded by the running shell.

Use this folder for short-lived design notes, reference snippets, or machine-specific material that is still being worked through. Completed plans, old reviews, and incident records belong in `docs/history/`, while current user or architecture documentation belongs in `docs/`.

Current reference files may include:

- `hyprland.lua` — an older/reference compositor configuration copy; do not treat it as the canonical live config without comparison.
- `power_icons.txt` — glyph/source reference for power-menu assets.
- `hypr-restructure/` — migration reference material associated with the completed Hyprland managed/user split.

Do not let this directory become a second backlog or revision history. Move durable decisions into the appropriate current document.
